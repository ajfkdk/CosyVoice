# 小说转视频完整工作流

自动将小说章节转换为配音视频的完整流水线。

## 功能特性

1. **16宫格生成** - 使用LLM提炼16个关键画面，生成故事板
2. **自动拆分** - 将16宫格拆分为独立场景
3. **HD升级** - 对每个场景进行2K高清升级，保持角色一致性
4. **语音合成** - 使用CosyVoice3生成说书人风格配音
5. **视频合成** - 自动合成最终视频，音画同步

## 核心特性

- ✅ 带重试机制的图片生成（5次重试，指数退避）
- ✅ 断点续传（中断后可继续执行）
- ✅ 完整日志记录
- ✅ 进度实时显示

## 使用方法

1. 将小说文本放入 `novels.txt`
2. 运行主流程：
```bash
python main.py
```

## 输出结构

```
output/
├── grid_16.png              # 16宫格原图
├── scenes/                  # 拆分的场景（原始）
├── scenes_hd/               # HD升级后的场景
├── audio/                   # 每个场景的音频
├── final_video.mp4          # 最终视频
├── progress.json            # 断点续传状态
└── pipeline.log             # 完整日志
```

## 依赖要求

- Python 3.8+
- ffmpeg
- CosyVoice3模型
- 参考音频文件
- 角色参考图

## 断点续传

如果流程中断，重新运行 `python main.py` 会自动：
- 跳过已完成的16宫格生成
- 跳过已完成的HD升级场景
- 跳过已完成的音频生成
- 继续未完成的部分
